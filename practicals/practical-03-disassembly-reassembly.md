# Practical 3 — Full Teardown & Rebuild

**Time:** 3 hours (5 phases with 3 trainer checkpoints + grading/cleanup)
**Work in pairs.** Swap roles (hands / documenter) at each phase.
**Student name:** ______________________  **Date:** ____________

**Golden rule for this entire practical: the battery is disconnected FIRST and reconnected LAST.**

## Phase 1 — Preparation (20 min)

| Step | Task | Done ✔ |
|---|---|---|
| 1.1 | Identify the exact model of your assigned phone (SIM tray / regulatory text) | ☐ |
| 1.2 | Find the ifixit (or equivalent) teardown guide and open it at your bench | ☐ |
| 1.3 | Read the guide once end-to-end BEFORE picking up a tool | ☐ |
| 1.4 | Build and test your ESD station (mat, strap, tester PASS) | ☐ |
| 1.5 | Photo 1: the intact phone, front and back, on the mat | ☐ |
| 1.6 | Lay out the magnetic project mat and label a grid zone per teardown stage | ☐ |

## Phase 2 — Opening (25 min)

| Step | Task | Done ✔ |
|---|---|---|
| 2.1 | Power the phone fully OFF (not sleep) | ☐ |
| 2.2 | Remove SIM tray and any external screws; map them | ☐ |
| 2.3 | Apply heat (heating pad or heat gun on low, keep moving) to soften adhesive | ☐ |
| 2.4 | Suction cup + picks to open a gap; picks only, shallow insertion near cameras and cables | ☐ |
| 2.5 | Open slowly like a book in the direction the guide shows — flex cables may tether the halves | ☐ |
| 2.6 | Remove the battery connector shield (map its screws) and disconnect the battery with a plastic spudger | ☐ |

**If you feel resistance while opening — stop. Something is still attached.**

### ✋ CHECKPOINT 1 — Battery disconnected before anything else

Trainer verifies the battery connector is lifted and NOTHING else was disconnected first.

## Phase 3 — Full teardown (45 min)

Strip the phone to a bare frame. Photograph each stage; every screw goes to its mapped grid zone.

| Step | Task | Done ✔ |
|---|---|---|
| 3.1 | Disconnect and remove the display assembly (if separate from frame) | ☐ |
| 3.2 | Remove all EMI/connector shields — one mat zone per shield, note screw lengths | ☐ |
| 3.3 | Disconnect all flex and coax cables (correct technique per connector type) | ☐ |
| 3.4 | Remove cameras, earpiece, sensors — store small parts in labelled zones | ☐ |
| 3.5 | Remove the motherboard — handle by edges only | ☐ |
| 3.6 | Remove battery (pull tabs or IPA + plastic card — NEVER a metal tool, never bend it) | ☐ |
| 3.7 | Remove sub-board, loudspeaker, vibration motor, buttons | ☐ |
| 3.8 | Photo each stage (minimum 5 stage photos) before moving to the next | ☐ |

### ✋ CHECKPOINT 2 — Bare frame + complete screw map

Trainer verifies: bare frame, every screw on the mat in a labelled zone, stage photos taken.

**Trainer signature:** ______________________  **Time:** ________

## Phase 4 — Reassembly (45 min)

Rebuild in exact reverse order using your photos and screw map.

| Step | Task | Done ✔ |
|---|---|---|
| 4.1 | Refit sub-board, loudspeaker, vibration motor, buttons | ☐ |
| 4.2 | Refit motherboard; seat it fully before any screws go in | ☐ |
| 4.3 | Reconnect every flex/coax; each press-fit must give a clean "click" — verify seating visually | ☐ |
| 4.4 | Refit cameras, earpiece, sensors, shields — correct screw from the correct zone every time | ☐ |
| 4.5 | Refit battery — connect it LAST, after all other connectors | ☐ |
| 4.6 | Cross-check: partner verifies every connector seated flat before shields close over them | ☐ |

**A half-seated connector can short and kill a board on power-up. Look, then look again.**

## Phase 5 — Test before sealing (25 min)

| Step | Task | Done ✔ |
|---|---|---|
| 5.1 | If the donor phone is functional: power on BEFORE gluing/closing; check display, touch, speaker, vibration, cameras | ☐ |
| 5.2 | If the donor is non-functional: trainer visual inspection of every connector seating instead | ☐ |
| 5.3 | Power off, close the phone, refit external screws | ☐ |
| 5.4 | Final photo: closed phone next to the EMPTY screw mat | ☐ |

### ✋ CHECKPOINT 3 — Reassembled, zero leftover screws

Trainer verifies: phone closed, mat empty, no leftover parts, test/inspection passed.
