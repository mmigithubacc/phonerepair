# Practical 6 — Ports & Small Parts

**Time:** 3 hours (4 stations × ~40 min + grading/cleanup)
**Work in pairs.** Rotate: one repairs, one documents; swap at each station.
**Student name:** ______________________  **Date:** ____________

**Rule for every station: DIAGNOSE before you replace. Prove the old part is faulty, prove the new one fixed it.**
**Battery disconnected first at every teardown. No metal tools inside a powered phone.**

## Station A — Charging port cleaning (35 min)

Cheapest repair in the shop — and the one most often sold as an expensive port replacement.

| Step | Task | Done ✔ |
|---|---|---|
| A1 | BEFORE reading: known-good cable + charger through the USB ammeter — record below | ☐ |
| A2 | Inspect the port with light + magnification; describe what you see: ______________________ | ☐ |
| A3 | Clean with a WOODEN toothpick or plastic pick — scrape lint from the back wall, never metal | ☐ |
| A4 | Short bursts of compressed air; if grime remains, a barely-damp IPA swab, then dry fully | ☐ |
| A5 | AFTER reading with the same cable/charger — record below | ☐ |
| A6 | Cable now clicks in with a firm, full-depth seat | ☐ |

| Measurement | Reading |
|---|---|
| Charge current BEFORE cleaning | ______ A |
| Charge current AFTER cleaning | ______ A |

**Function-test sign-off (trainer):** ______________________

## Station B — Charging sub-board replacement (50 min)

On most Androids the port lives on a sub-board that also carries the primary mic and
loudspeaker contacts — so a sub-board swap must be retested for ALL THREE.

| Step | Task | Done ✔ |
|---|---|---|
| B1 | Confirm the fault justifies replacement (cleaning at Station A logic first) | ☐ |
| B2 | Guide open; teardown to the sub-board; battery disconnected FIRST; screws mapped | ☐ |
| B3 | Disconnect the main flex/board-to-board interconnect from the sub-board | ☐ |
| B4 | Remove old sub-board; compare it to the new part side by side (same connectors, same mics/contacts) | ☐ |
| B5 | Fit new sub-board; seat interconnect and any coax fully; reassemble; battery LAST | ☐ |
| B6 | Retest ALL THREE: ammeter charge reading ______ A ▸ mic (voice-recorder test) ▸ loudspeaker (ringtone at full volume) | ☐ |

**Function-test sign-off (trainer) — charge / mic / speaker all pass:** ______________________

## Station C — Loudspeaker replacement (35 min)

| Step | Task | Done ✔ |
|---|---|---|
| C1 | Pre-test: play a ringtone — describe the fault (silent / crackle / muffled): ______________ | ☐ |
| C2 | Open, battery disconnected FIRST, remove the speaker module | ☐ |
| C3 | Check spring contacts on board and module — clean with IPA if oxidised | ☐ |
| C4 | Fit new speaker; do not pinch any flex under the module | ☐ |
| C5 | Sound test before final closing: ringtone at full volume — loud and clean, no buzz | ☐ |
| C6 | Close up; repeat the sound test; zero leftover screws | ☐ |

**Function-test sign-off (trainer):** ______________________

## Station D — Button flex OR camera swap (40 min)

Trainer assigns per available parts: power/volume button flex, or front/rear camera module.

| Step | Task | Done ✔ |
|---|---|---|
| D1 | Pre-test and record the exact symptom: ______________________ | ☐ |
| D2 | Open, battery disconnected FIRST, teardown to the assigned part | ☐ |
| D3 | Note the cable routing (photo!) — button flexes snake under boards and shields | ☐ |
| D4 | Swap the part; re-route the flex exactly as photographed; seat connectors fully | ☐ |
| D5 | Function test before sealing: every button clicks & registers / camera focuses, no dust in image | ☐ |
| D6 | Reassemble; retest; zero leftover screws | ☐ |

**Function-test sign-off (trainer):** ______________________
