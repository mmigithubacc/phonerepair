# Practical 4 — Screen Replacement

**Time:** 3 hours (7 phases + grading/cleanup)
**Work in pairs.** One repairs, one documents & cross-checks; swap on the next phone if time allows.

**The #1 rule of this repair: TEST BEFORE SEALING. Glue nothing until the new screen passes every test.**

## Phase 1 — Prep & guide lookup (15 min)

| Step | Task | Done ✔ |
|---|---|---|
| 1.1 | ESD station built and strap tested (PASS) | ☐ |
| 1.2 | Exact model identified; screen-replacement guide open at the bench | ☐ |
| 1.3 | Inspect the NEW screen out of its box: no cracks, protective films intact, correct part | ☐ |
| 1.4 | Photo 1: the phone with the old/broken screen, powered on if possible | ☐ |
| 1.5 | Note where the display cables run on this model — plan your pick depth | ☐ |

## Phase 2 — Remove the old screen (35 min)

| Step | Task | Done ✔ |
|---|---|---|
| 2.1 | Power off. If the glass is shattered, tape over it before applying suction | ☐ |
| 2.2 | Remove external screws; start the screw map | ☐ |
| 2.3 | Heat the edges (low, keep the heat moving) to soften adhesive | ☐ |
| 2.4 | Suction cup to lift, pick into the gap; shallow insertion only near cameras and cable runs | ☐ |
| 2.5 | Open like a book toward the cable side — the old screen is still tethered | ☐ |

**A pick pushed too deep cuts flex cables and turns a screen job into a board job.**

## Phase 3 — ✋ Battery disconnect checkpoint

| Step | Task | Done ✔ |
|---|---|---|
| 3.1 | Remove the connector shield; disconnect the BATTERY first with a plastic spudger | ☐ |
| 3.2 | Only now disconnect the display/touch/sensor cables | ☐ |

## Phase 4 — Transfer small parts (25 min)

Check the guide: on your model, which parts stay on the frame and which must move to the new screen?

| Step | Task | Done ✔ |
|---|---|---|
| 4.1 | Transfer earpiece mesh / speaker bracket if the new screen lacks them | ☐ |
| 4.2 | Transfer sensor holders / proximity sensor bracket as applicable | ☐ |
| 4.3 | Transfer any mounting plates or shields screwed to the old display | ☐ |
| 4.4 | Peel the old adhesive from the frame; clean the groove with IPA | ☐ |
| 4.5 | Lay transferred parts photo next to the new screen — Photo 2 | ☐ |

## Phase 5 — Connect & TEST BEFORE SEALING (35 min)

Connect display cables, reconnect battery LAST, power on with the phone still open. Run the full protocol:

| Test | How | Pass ✔ | Notes |
|---|---|---|---|
| Touch — every zone | Drawing app; scribble to all four corners and edges, no dead stripes | ☐ | |
| Dead/stuck pixels | Full-screen white, black, red, green, blue test images | ☐ | |
| Brightness sweep | Slide 0 → 100%; no flicker, no dark patches | ☐ | |
| Auto-brightness / light sensor | Cover the sensor; brightness responds | ☐ | |
| Proximity sensor | Place a call, cover the earpiece area — screen must switch off | ☐ | |
| Earpiece audio | Call audio clear through the transferred mesh | ☐ | |
| Front camera window | Open camera; image sharp, window clean, no fingerprints inside | ☐ | |

**Any failure: power off, disconnect battery, reseat cables, retest. Do NOT proceed to adhesive.**

## Phase 6 — Adhesive & closing (25 min)

| Step | Task | Done ✔ |
|---|---|---|
| 6.1 | Power off and disconnect the battery again before final fitting | ☐ |
| 6.2 | Apply pre-cut adhesive or a thin B-7000 bead — no glue near cables, speakers, or mics | ☐ |
| 6.3 | Reconnect all cables, battery last; refit shields with mapped screws | ☐ |
| 6.4 | Seat the screen edge-first on the cable side, then lower flat; even finger pressure around the perimeter | ☐ |
| 6.5 | Refit external screws; clamp or weight the phone if using liquid glue | ☐ |

## Phase 7 — Final QC (15 min)

| Step | Task | Done ✔ |
|---|---|---|
| 7.1 | Repeat quick tests: touch corners, earpiece, proximity | ☐ |
| 7.2 | Check gap lines all round — screen sits flush, no lifted corners | ☐ |
| 7.3 | Photo 3: the finished phone powered on (before/after pair with Photo 1) | ☐ |
| 7.4 | Screw mat empty; bench cleaned; old screen to the correct waste bin | ☐ |
