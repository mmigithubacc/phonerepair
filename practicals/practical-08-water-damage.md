# Practical 8 — Water Damage Treatment

**Time:** 3 hours (9 protocol phases + customer report + grading/cleanup)
**Work in pairs.** Each pair receives one pre-corroded donor phone (trainer salt-water dunked it days ago — treat it as a real customer job).
**Student name:** ______________________  **Date:** ____________

**Prime directive: liquid + power = dead board. The phone stays OFF and the battery comes OUT early.**
**A water-exposed battery is NEVER reused — it goes to quarantine.**

| Step | Task | Done ✔ |
|---|---|---|
| 1.1 | Roleplay intake with your partner as customer: what liquid, when, was it powered/charged since? Record answers below | ☐ |
| 1.2 | DO NOT attempt to power the phone on | ☐ |
| 1.3 | Check the Liquid Contact Indicators (SIM slot, headphone jack, internal per model) — colour: __________ | ☐ |
| 1.4 | Photo 1: exterior + visible LCI | ☐ |

Intake notes: liquid ______________ , exposure time ______________ , powered since? ______

## Phase 2 — Teardown & battery quarantine (25 min)

| Step | Task | Done ✔ |
|---|---|---|
| 2.1 | ESD station tested; open the phone (heat sparingly — trapped moisture can flash to steam) | ☐ |
| 2.2 | Disconnect the battery FIRST | ☐ |
| 2.3 | REMOVE the battery entirely — water-exposed battery goes to the quarantine tray, labelled | ☐ |
| 2.4 | Strip to the board level: shields off where the guide permits, board out, handle by edges | ☐ |
| 2.5 | Photo 2: board as found, before any cleaning | ☐ |

**Trainer check — battery in quarantine tray: ______**

## Phase 3 — Magnified inspection & corrosion map (20 min)

| Step | Task | Done ✔ |
|---|---|---|
| 3.1 | Inspect the whole board under magnification, both sides, and under/around shields | ☐ |
| 3.2 | On the printed board sketch (trainer provides), circle and number EVERY corrosion site: green/white fuzz, dark tracks, dull connectors | ☐ |
| 3.3 | Count of sites found: ______ — pay special attention to connectors and around the battery terminals | ☐ |
| 3.4 | Photo 3: worst corrosion site, close up | ☐ |

## Phase 4 — IPA brush clean (20 min)

| Step | Task | Done ✔ |
|---|---|---|
| 4.1 | ≥ 99% IPA + soft ESD brush; scrub each marked site in small circles | ☐ |
| 4.2 | Brush connectors gently along, not across, their pins | ☐ |
| 4.3 | Work through your numbered sites in order — tick each on the sketch as cleaned | ☐ |

## Phase 5 — Ultrasonic cycle (20 min)

| Step | Task | Done ✔ |
|---|---|---|
| 5.1 | BOARD ONLY in the tank — never the battery, display, speakers, or cameras | ☐ |
| 5.2 | Correct solution (dedicated PCB fluid, or ≥ 99% IPA per shop policy) at the marked fill line | ☐ |
| 5.3 | Suspend the board in the basket, not touching the tank floor | ☐ |
| 5.4 | Run the timing the trainer specifies for this tank: ______ min — do not "double it to be sure" | ☐ |

## Phase 6 — Dry (15 min)

| Step | Task | Done ✔ |
|---|---|---|
| 6.1 | Shake off, blot edges, then warm-air dry (low heat / drying chamber) — patience, not a hot heat gun | ☐ |
| 6.2 | Confirm connectors and under-shield areas are bone dry before any reassembly | ☐ |

## Phase 7 — Post-clean inspection (15 min)

| Step | Task | Done ✔ |
|---|---|---|
| 7.1 | Re-inspect every numbered site under magnification — clean metal, or damage remaining? Note per site on the sketch | ☐ |
| 7.2 | Photo 4: same worst site as Photo 3, after cleaning | ☐ |
| 7.3 | Note any eaten-through pads/traces — these are refer-to-microsoldering findings, not failures | ☐ |

## Phase 8 — Reassemble, test & verdict (20 min)

| Step | Task | Done ✔ |
|---|---|---|
| 8.1 | Reassemble with a KNOWN-GOOD battery from the trainer (quarantined battery stays quarantined) | ☐ |
| 8.2 | Power on; test display, touch, charge (ammeter: ______ A), speakers, cameras, buttons | ☐ |
| 8.3 | Verdict: fully working / partially working (list what's dead) / no power — refer to board-level | ☐ |
