## 3. Final grade calculation

Weights per the course overview: quizzes 20% • practical lab rubrics 30% • written final 20% • practical final 30%.

| Component | Raw score (%) | Weight | Weighted points |
|---|---|---|---|
| Module quiz average (10 quizzes) | | × 0.20 | |
| Practical lab rubric average (Practicals 1–9) | | × 0.30 | |
| Final written exam | | × 0.20 | |
| Final practical exam | | × 0.30 | |
| **FINAL GRADE** | | **1.00** | **____ %** |

**Pass rule:** final grade **≥ 70%** AND final practical exam **passed** (≥ 70 with zero critical violations). A student with a 70%+ average who failed the practical final is NOT certified until they pass a practical retake.

### Worked example

| Component | Raw score | Weight | Weighted |
|---|---|---|---|
| Quiz average | 82% | × 0.20 | 16.4 |
| Lab rubric average | 75% | × 0.30 | 22.5 |
| Written final | 71% | × 0.20 | 14.2 |
| Practical final | 80% (passed, no violations) | × 0.30 | 24.0 |
| **Final grade** | | | **77.1% → PASS** |

Grade bands (optional, for the certificate): 70–79 Pass • 80–89 Merit • 90–100 Distinction.
